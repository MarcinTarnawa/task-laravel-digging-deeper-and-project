# Laravel: Digging deeper and project (24-30)

Wykorzystaj system generowania faktur wraz z uwierzytelnianiem z [poprzedniego projektu](https://github.com/devmentor-pl/task-laravel-authentication).

Tym razem rozbuduj go o kilka dodatkowych funkcjonalności:
- możliwość wysłania faktury na maila klienta (dodaj niezbędne dane) jako załącznik
- możliwość personalizacji PDF-a o:
    - logo tj. zdjęcie wgrane na serwer i przypisane do użytkownika
    - ustawienie kolorystyki np. kolor tła, kolor tekstu, kolor obramowania tabel zdefiniowane w panelu użytkownika
- przygotuj mechanizm, który ograniczy możliwość rozsyłania spam-u i jednocześnie będzie zapobiegać pomyłką. Zanim faktura zostanie wysłana, użytkownik który ma otrzymać fakturę musi wyrazić na to zgodę. Może to polegać na kliknięciu w link w wiadomości email, która jest wysyłana tylko przy pierwszej próbie wysłania faktury.
